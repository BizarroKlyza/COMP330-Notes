W2 Lecture 1

GPUs are very good at doing lots of individual things very very quickly. Not so much complex calculations.
CPU or GPU for the following instructions?:

  • execute a stream of simple instructions as quickly as possible
    • cpu
  • compute the same things with many pieces of data at the same time
    • gpu
  • transform 1000 vertices
    • gpu
  • compute 20!
    • cpu
  • compute the colours of 10000 pixels
    • gpu
  • execute a shader program on 5,000 vertices
    • gpu

Vectors:

  • a mathematical object that has a length and direction but no starting point (no specified origin)
  • a vector can be moved and still maintain the same value regardless of position.
  • if we want to add two vectors together:
    • we put the tail of one vector at the head of the other, and then the sum is the vector that goes from the tail of the first vector to the head of the second.



When we add the vectors we add the corresponding elements of them together, ie. add the x values together and the Y values together.

If we multiply (scalar) essentially increases the length of the vector (in x and y) without changing direction.

We can also compute the vector between two points.


This is infact the same thing as subtracting one vector from another, as points can be thought of as vectors from the origin to the point.

We can also define a line through two points.
We have two vectors A and B that we have placed at the origin so that they represent points.
We want a line that goes through these two points.
A parametric line equation is an equation where we have a value T that changes as we move through the line.
Ie:


We want to find an equation for point P such that it is on the line.
We know that if T = 0, it should be at A
If T = 1, we want P = B.


We define the magnitude of a vector as its length.(vectors are normally written as Boldface letters (ie. a) while scalars are written as Italics (ie. _t_).
The magnitude of a vector (length) = |a| = √ax^2 + ay^2
This is the euclidean length of the vector.

A unit vector is a vector where it’s length is equal to 1, pointing in the same direction as another vector.
We can normalise a vector and get its unit vector, where the unit vector in the direction of a = the vector A, divided by the length of the vector A

The next useful thing is the dotproduct.
This is equal to the product of all corresponding elements then add them all up. The length of a vector A is equal to the dotproduct of A with itself, squarerooted.


Matrices

An object that has some numbers in it that when we multiply by a vector we get another vector.
The rule for multiplication here is that we go row by column.


polygons:

A useful way of drawing shapes.
Webgl breaks polygons down into triangles called a mesh.
one type of poly is a convex poly, where all of the angles go in the same direction, with no concave angles.
